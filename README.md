## Условие
Написать программу, которая из имеющегося массива строк формирует массив строк, длина которых меньше либо равна 3 символа.
При решении не использовать коллекции.

## Решение
#### Ввод строк исходного массива
Ввод строк осуществляется через консоль.<br/>
Если пользователь пытается ввести пустую строку, она в массив не добавляется.<br/>
Если пользователь вводит не пустую строку, она добавляется в массив.<br/>
Далее система запрашивает у пользователя, желает ли он ввести еще одну строку (Y/n).<br/>
Если пользователь отвечает утвердительно (Y), запрашивается ввод еще одной строки.<br/>
Если пользователь отвечает отрицательно (n), система переходит к следующему этапу работы (формированию массива строк согласно условию).<br/>
Если пользователь вводит ответ, отличный от утвердительного или отрицательного, система повторяет вопрос.

#### Добавление строк в массив
По условию не допускается использовать коллекцию (List), что упростило бы решение задачи (мы могли бы добавить введенные пользователем
строки в список List, а затем преобразовать List в массив). <br/>
Поэтому будет использоваться метод Linq Concat() для объединения элементов двух входных последовательностей. <br/>
Этот способ будет использоваться и для первого массива (который создается на основе введенных пользователем строк), и для второго массива,
который формирует система по заданному условию.

#### Формирование массива по заданному условию
После того, как пользователь завершил ввод строк и система сформировала первый массив, система переходит к последовательной проверке длины строк 
первого массива.<br/>
Мы могли бы использовать оператор foreach для перебора всех элементов первого массива, но для наглядности используем последовательный перебор 
каждого элемента массива по его индексу.<br/>
Для этого вначале определяем длину массива.<br/>
И последовательно берем каждый из элементов массива по индексу, начиная с индекса = 0, до тех пор, пока индекс меньше длины.
Вычисляем длину строки выбранного элемента.<br/>
Если длина строки меньше или равна трем, добавляем данный элемент во второй массив.<br/>
После того, как обход всех элементов первого массива завершился, выводим в консоль значения, которые мы поместили во второй массив.
